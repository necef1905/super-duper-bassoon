# Math Wallet

## What is it?

Math Wallet is the most powerful and secure universal crypto wallet that enables storage of all BTC, ETH/ERC20, NEO/NEP5, EOS, TRON tokens, supports cross-chain token exchange and a multi-chain dApp store.
